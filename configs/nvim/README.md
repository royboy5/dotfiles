# NVIM

This is the NVIM readme.

## nvim directories 
config: `~/.config/nvim`
data: `~/.local/share/nvim/`
plugins: `~/.local/share/nvim/site/pack/packer/`
logs: `~/.cache`
state: `~/.local/state`
